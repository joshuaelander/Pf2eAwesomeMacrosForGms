# Pf2eAwesomeMacrosForGms
Overview: A collection of very useful Foundry VTT macros for GMs to use in PF2e
Contents:
- Random Encounter Creator
- Full party heal and status restore (for dungeon healing/full rest)
- Auto grant XP with notice of level ups
- Many more to come!
